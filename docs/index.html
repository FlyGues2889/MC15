耗时一年的巨大工程

我们在毕业前不久便一直有为母校建设一个MC版本的想法，最终这个想法变为项目并在 2023年10月 正式立项。在广泛搜集各种关于学校的照片等资料后，复刻工程在2023年11月4日开始。

经过在学余时间不断修补，复刻工程已经基本于2024年8月2日正式结束。


修改 100,000+ 个方块
新增 1,000+ 个实体
花费 500+ 小时建造

第一个正式版于 2024年08月24日 正式发布，并在网站上开放下载。在此之后，我们会继续完善项目，并将存档部署到网站上，让更多人能够访问到复刻工程的成果。

建筑主要使用 Axiom 模组 帮助建造，其为项目的提早完成提供了极大的帮助。
